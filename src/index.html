This project features a file for Sticky Nav (js/sticky-nav.js) which has follwing features:

  1. Make a nav bar sticky based on scroll event
  2. Can be made into a seperate component so that it could install using npm
  3. Future iteration will use intersect observer for better performance

The cross.scss file contains the logic to create a plus sign using CSS

  1. This can we used by using the html <div class="plus-container closed"><div class="plus"><div class="horizontal"></div><div class="vertical"></div></div></div>
  2. Class opened can be toggled to see the desired effect

Commands to Run this project:

  • npm start
  • npm run build-css (to compile the common css file)
  • Server is going to run port 8080
